### Навигация

- Нажмите на имя папки для перехода в неё
- Используйте хлебные крошки вверху страницы для быстрой навигации
- Нажмите ".." для возврата в родительскую папку

### Поиск файлов и папок

1. Введите имя файла или папки (минимум 2 символа) в поле поиска на панели инструментов
2. Нажмите кнопку "Search" или нажмите Enter
3. Результаты отобразятся в виде списка с информацией:
   - Имя файла/папки
   - Тип (файл или папка)
   - Полный путь к элементу
4. Кликните по любому результату для перехода к папке, содержащей найденный элемент
5. Нажмите ✕ для закрытия результатов поиска

**Особенности поиска:**
- **Поиск в текущей директории** и всех её подпапках
- Регистронезависимый поиск
- Поиск по частичному совпадению имени (формат: `*искомый_текст*`)
  - Пример: поиск "test" найдет "test.php", "mytest.txt", "new_test_file.js"
- Максимум 100 результатов
- Автоматически пропускает системные папки (.git, cache, tmp, logs)

**Расширенный поиск с фильтрацией по дате:**
1. Нажмите кнопку 🔍 "Advanced Search" рядом с полем поиска
2. Откроется панель с дополнительными фильтрами:
   - **Изменено после** - найти файлы изменённые после указанной даты
   - **Изменено до** - найти файлы изменённые до указанной даты
3. Выберите даты и выполните поиск
4. Используйте кнопку "Clear Filters" для сброса фильтров

*Примеры использования:*
- Найти все файлы изменённые за последнюю неделю
- Найти файлы созданные до определённой даты
- Комбинировать поиск по имени и дате (например, все ".log" файлы за январь)

### Изменение прав доступа (chmod)

1. Нажмите кнопку 🔒 (Permissions) для файла или папки
2. В открывшемся окне введите права в восьмеричном формате:
   - `0644` - стандартно для файлов (владелец: чтение+запись, остальные: чтение)
   - `0755` - стандартно для папок (владелец: все права, остальные: чтение+выполнение)
   - `0777` - полные права для всех (не рекомендуется)
3. Нажмите "Apply"

**Формат прав доступа:**
- Первая цифра (обычно 0) - специальные биты
- Вторая цифра - права владельца (4=чтение, 2=запись, 1=выполнение)
- Третья цифра - права группы
- Четвёртая цифра - права остальных

Примеры:
- `0644` = rw-r--r-- (файлы)
- `0755` = rwxr-xr-x (папки, скрипты)
- `0600` = rw------- (конфиденциальные файлы)

**Для файлов отображается:**
- Полное имя файла
- Абсолютный путь
- Размер в удобном формате (Bytes, KB, MB, GB)
- MIME-тип (тип содержимого)
- Дата и время последнего изменения
- Текущие права доступа

**Для папок отображается:**
- Имя папки
- Абсолютный путь
- Количество файлов внутри
- Количество подпапок
- Дата и время последнего изменения
- Текущие права доступа
